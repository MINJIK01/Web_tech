Testing

This page shows the test result of this webpage.

Optimisation

To shorten the image loading time, the pixels of the image were reduced to an appropriate size.

Debugging

First, in the css file, there was no error. However, the result showed me warning sign and it was because there was the same color between background-color and color. But it was my intention for hiding answer at the quiz page so it did not matter.

Continually, I checked html files and then I could find this error at the all html files. This error occured because there were another tags before body tag. Thus, I changed location of tags properly into the body tag and the error disappeared.

2. Edge

Accessibility

As a result of the test, errors related to language setting and contrast were found. Therefore, I set HTML's language to English so that it can be applied in the appropriate language when using the screen reader. Also, I changed the color of footer link to black to solve the contrast issue.
