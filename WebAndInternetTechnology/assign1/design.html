Design

This page explains design of this webpage.

Introduction

This website was created with the goal of giving a cheery and joyful image. The motif of it is cherry blossoms, and the icon representing the website is set to cherry blossoms so that users can recognize the motif immediately. In addition, the overall color of the website was set to pink to give the feeling of cherry blossoms, and only differenciate the concentration of pink, so that the color could be used to give a sense of unity.

Site Map

The structure of the website is quite simple. The homepage is main page and can be moved to all another submenu, and each page was also accessed to another page.

Design Mock Ups

I used one break point. The reason is that when the area was reduced, the minimum area where pictures and writings could be located without exceding the div box in columns was 800px. In addition, to make elements arranged in columns are placed in a row when the screen is reduced, pages are used flex container. The Degree program page is designed with the same as the home page. The SGA and mySkills pages and Quiz pages did not place elements in the div box as columns, but were placed in rows from the beginning, so the basic design was used without using a flex container.

Menu System

The menu bar is designed to be stretched in a row. However, in this case, when the screen becomes smaller, the menu goes out of the screen and is not visible. Therefore I made the items on the menu can be arranged in row direction when the screen becomes smaller using a flex container.
