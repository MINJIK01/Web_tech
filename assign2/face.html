This program is for drawing face.

If user chooses anger button, then eyebrows' color is changed and also their shape is changed. In addition, the mouth's shape is changed as well.

If user chooses neutral button, then it is returned to the first facial state.

If user chooses happiness button, then eyebrows are raised and the mouth's shape is changed. In addition, flushed cheeks are appeared.

To explain additional functions, if user clicks an eye (the upper part of the eye), then it closes eye. Also, if user clicks closed eye agian, then it opens its eye with being bloodshot. When user clicks its eye more than 5 times, its expression state is changed as an anger expression. If user continues clicking eye even though it is anger state, then it shouts to user! In additon, if user clicks the upper center of the mouth, then mouth color is changed.
